Description
Aggregate Rss Feeds and print Ticker Text to a handle. Requires extra/haskell-http and aur/haskell-tagsoup packages.

Usage

To use, add feed urls (one per line, no whitespace) to a file at ~/.rssfeeds then copy the source code for this module into ~/.xmonad/lib/RssReader.hs, then add the following to your ~/.xmonad/xmonad.hs: 

 import RssReader
 import XMonad.Util.Run (spawnPipe)

 main :: IO()
 main = do
     spawnPipe "dzen2 -p -tw 100" >>= spawnReader defaultReaderConf

     xmonad $ defaultConfig
         { ...
         , ...
         }
