Usage

To use, copy the source code for this module into ~/.xmonad/lib/Dzen.hs and add the following to your ~/.xmonad/xmonad.hs: 

 import Dzen
 import XMonad.Hooks.DynamicLog

 main :: IO ()
 main = do
     d <- spawnDzen defaultDzen

     xmonad $ defaultConfig
         { ...
         , logHook = myLogHook d
         }

 myLogHook h = dynamicLogWithPP $ defaultPP
     { ...
     , ppOutput = hPutStrLn h
     }

If you just want to spawn a dzen and don't care about its handle you can use the following: 

 spawn $ show defaultDzen
